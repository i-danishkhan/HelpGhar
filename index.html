HelpGhar

A web-based platform connecting customers with verified home service providers: maids, drivers, plumbers, electricians, and tutors.

Microservice Architecture 3-Tier Client-Server Web Platform

HelpGhar is a web-based service platform designed to connect customers with verified home service providers. The system facilitates end-to-end discovery, booking, payment, and review of skilled workers across domestic and professional categories including maids, drivers, plumbers, electricians, and tutors.

System Architecture
Architecture Layers
Presentation

Web Portal rendered on Android and Desktop clients for all user roles.

Application

API Gateway managing six independent backend microservices: Authentication, Booking, Payment, Tracking, Recommendation, and Administration.

Data

Dedicated databases per microservice ensuring isolation and independent scalability.

What It Does
Core Features
01

Dual-Role Accounts

A single account can toggle seamlessly between Customer and Worker mode.

02

Verification & Onboarding

Multi-step worker onboarding with CNIC and certificate uploads, followed by admin approval.

03

Booking & Scheduling

Real-time availability checks and conflict-free scheduling to prevent double-bookings.

04

Payment & Escrow

SSL-encrypted pipeline supporting cards and digital wallets (Google Pay, JazzCash) via escrow model.

05

Work Tracking

Real-time job status updates with automated performance-based reward system.

06

Smart Recommendations

Algorithmic worker suggestions based on location, skill, and average ratings.

07

Admin & Disputes

Centralised control panel for user governance, dispute history, and platform analytics.

Who Uses It
System Users
CUS

Customer

Browses categories, views worker profiles, books services, and makes secure payments.

WRK

Worker

Manages service listings, updates job status, and tracks earnings and incentive points.

ADM

Administrator

Verifies workers, resolves disputes, issues refunds, and monitors system-wide analytics.

Protection
Security & Constraints

Data Protection

Enforced SSL/TLS encryption for all client-server communication to protect sensitive financial data.

Access Control

Role-Based Access Control (RBAC) ensures users only access features within their authorised scope.

Reliability

High-availability infrastructure with daily automated cloud backups to secure storage.
